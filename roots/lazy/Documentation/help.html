Getting Startedthings to know before get started

We assume you have basic knowledge of HTML / CSS and JS. In this doc you will not learn how to code but you will learn how to setup, use and customize the theme. Before you begin make sure you have a code editor, you can use Dreamweaver or any other editor like Notepad++ will be enough.

File structure

Within the download you'll find the following file structure, logically groupped common assets.

Once downloaded, unzip the compressed folder to see the structure of the template. You'll see something like this:

  LazyGuy/
  ├── css/
  ├── js/

  └── other html files

Put Your Content

In this template you will find all HTML files in well formatted and highly commented. Most of the markup is self explanatory with comments. So i will only explain the parts where the markup needs more attention.

Change Background Image

You can easily change background image with some simple steps;

1 Prepare your image; resize it to 1920px width, save as .jpg file (quality 70%) for the optimal image size to have a fast loading page.

2 Copy your image to /images folder.

3 find this code and change image url in the markup;

<!-- BG IMAGE / VIDEO -->
<div class="media" style="background-image:url(images/your-image.jpg)"></div>

Change Background Video

You can easily change background video. Just copy your youtube or vimeo video url and replace it with the one below;

<!-- BG IMAGE / VIDEO -->
<div class="media" data-jarallax-video="https://vimeo.com/122322480"></div>

Customizationhow to customize the template to your needs.

We designed this template for easy customizations. It includes predefined classes so you can easily customize the template just by adding some classes.

Body Classes

Here are some classes you can add to body tag for various customizations;

  • is-text-light : makes body text light for pages with dark backgrounds.
  • is-text-align-center : center aligns body text (body text is aligned left by default)
  • is-text-align-right : right aligns body text
  • is-medium-typo : body with smaller typohraphy.
  • is-small-typo : body with much smaller typohraphy.
  • is-masked-light : applies light masking on image or video - to use it on light backgrounds.
  • is-masked-dark : applies dark masking on image or video - to use it on dark backgrounds.
  • is-masked-gradient-blue : applies blue-gradient masking on image or video.
  • is-masked-gradient-pink : applies pink-gradient masking on image or video.
  • is-masked-gradient-purple : applies purple-gradient masking on image or video.
  • is-masked-gradient-red : applies red-gradient masking on image or video.
  • is-masked-gradient-green : applies green-gradient masking on image or video.

Custom Fonts

Using Custom Fonts by Google Fonts

You can customize fonts with this simple steps;

1 Go to Google Fonts and select a font.

2 Copy the standard embed code for your font and paste it at the end of head tag

<link href="https://fonts.googleapis.com/css?family=Roboto+Mono:400" rel="stylesheet">

3 Finally add this custom css code after the font embed code in head tag;

<!-- CUSTOMIZATONS -->
<style>
	
	<!-- HEADING FONT -->
	h1 { font-family: 'Roboto Mono', monospace; font-weight: 400; }
	
	<!-- BODY FONT -->
	body { font-family: 'Roboto Mono', monospace; }
	
</style>
