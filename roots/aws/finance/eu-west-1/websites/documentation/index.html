Leza - React Documentation

Introduction

Leza is a Retina Ready Responsive React NextJS template for Funding, Loans and Banking Agencies. It has 3+ beautiful demo variations with inner pages. The template is developed with ReactJS, Next.js, Bootstrap 5.x, and Sass. This best fit for Loan Provider, Financial Companies, Banking website, Payment Gateway provider, Financial Agency, Online Money Recharge or Send + Receiving Money website, Financial Agency, Digital Banking website, etc.

Setup Development Environment

To kickstart the development of the web app with Leza, at first need to setup the react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site https://nodejs.org/en/

Preparation

You'll need to install Node.js >=v14.15+ (Recommended Version) (NPM comes along with it) and serve (an NPM package)

Use command line tool i.e. PowerShell or terminal and navigate to the project(Leza) root. This is PowerShell in windows and terminal on mac/linux.

Let's have a look on the setup process:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run dev

It'll return something like this in the console:

> project@1.5.0 dev drive:\project

> next

[ wait ] starting the development server ...

[ ready ] started server on http://localhost:3000 ...

[ event ] compiled successfully

Now, in the browser go to localhost:3000

For Production Build

Run : npm run build

Then Run : npm run start

See the official next.js doc here

Fonts Used

By default, the template loads Oswald & Open Sans font from Google Web Font Services, you can change the font with the one that suits you best. Google Fonts

You will find the font code in the "style.scss file: styles/style.scss


@import url('https://fonts.googleapis.com/css2?family=Rubik:ital,wght@0,300;0,400;0,500;0,700;0,900;1,300;1,400;1,500;1,700;1,900&display=swap');

@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');

$main-font-family: 'Roboto', sans-serif;
$heading-font-family: 'Rubik', sans-serif;

Note: After SCSS file run then it will be work
